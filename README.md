# 📊 Student Productivity Analysis

## 📌 Overview

This project explores the key factors influencing student productivity using a dataset of 5,000 students. The goal is to uncover meaningful patterns between study habits, mental health, and academic performance through data analysis and visualization.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze relationships between productivity and lifestyle factors
* Generate insight-driven visualizations
* Build a simple analytical dashboard

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

The dataset contains features such as:

* Age, Gender
* Study hours, Sleep hours
* Screen time, Social media usage
* Mental health score
* Burnout level
* Focus index
* Productivity score
* Exam score

---

## 🔍 Key Analysis Performed

### ✔ Data Cleaning

* Handled missing values using median (numeric) and mode (categorical)
* Ensured consistent column formatting

### ✔ Exploratory Data Analysis (EDA)

* Distribution analysis of productivity
* Relationship between study hours and productivity
* Impact of burnout on mental health
* Gender-based comparisons

### ✔ Feature Relationships

* Pairplot to visualize interactions between variables
* Correlation heatmap to identify strong relationships

---

## 📈 Key Insights

* **Focus index** has a strong positive correlation with productivity
* **Burnout** negatively impacts both mental health and productivity
* **Study hours** improve productivity up to a certain limit
* Balanced sleep and lower screen time contribute to better outcomes
